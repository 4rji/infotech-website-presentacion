AI-Driven Backdoor Detection System

Real-Time Threat Detection for Modern Networks

By:

Havi Lead Security Engineer 2025 Capstone Project

Overview

AI-based, open-source tool for detecting and blocking network backdoors in real time.

  • Detects anomalies using machine learning
  • Scalable and modular design
  • Real-time threat response
  • Easy to deploy on low-cost hardware
  • Integrates with existing security tools

Problem Statement

Traditional network security tools struggle to detect hidden threats like backdoors, especially in real time.

Many organizations lack the resources or expertise to deploy complex, high-cost security systems.

This project addresses that gap with an accessible, automated, and intelligent solution.

Goals & Objectives

The project aims to improve threat detection and simplify response through AI.

  • Detect and block threats in real time
  • Ensure easy installation and low technical barrier
  • Enable integration via flexible APIs
  • Support both small and enterprise-scale environments
  • Deliver a clear, user-friendly monitoring interface

Scope Statement

Included:

  • AI engine for detecting network anomalies
  • Real-time log and traffic analysis
  • Web and mobile monitoring interfaces
  • Integration with tools like Suricata and Splunk
  • Deployment on devices like Raspberry Pi 5

Excluded:

  • Manual threat analysis workflows
  • Proprietary or commercial software components
  • On-premise training of large AI models
